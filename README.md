## 🗺️ Core Surveying & Engineering Capabilities

1. **Construction Site Land Surveying**: High-accuracy leveling, structural column grid layout, benchmark elevation setting, and earthwork cut/fill volume computations.
2. **Boundary Demarcation & Pillar Fixing**: Revenue map (`Shajra`/`Musavi`) verification, boundary encroachment resolution, and DGPS concrete corner pillar installation.
3. **Topographical & Contour Mapping**: Detailed 3D surface elevation profiles, digital elevation models (DEM), slope analysis, and municipal clearance drawings.
4. **Railway Track & Highway Alignment**: Specialized corridor alignment, horizontal/vertical curve calculations, L-sections, and X-sections for transport networks.
5. **Agricultural & Garden Land Surveys**: Precise acreage/bigha calculation, family property sub-division, hill terracing layouts, and water drainage profiling.
6. **Pond Surveying & Airport Land Processing**: Reservoir depth profiling, silt accumulation storage calculations, Obstacle Limitation Surface (OLS), and hydrographic mapping.

## 🚀 Local Development & Preview

No Node.js build steps or heavy bundlers required. Simply run any local static server:

```bash

npx serve .

# Or using Python built-in HTTP server
python -m http.server 3000
```

Visit `http://localhost:3000` in your web browser.
